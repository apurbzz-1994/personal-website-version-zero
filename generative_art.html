I make artwork and animations with code using the Processing programming language, and sometimes for the web using P5.JS.

  • Typography
  • Animations based on simple physics engine coded in Processing with vectors, forces, friction etc.
  • Animations that respond to changes in surroundings via sensors (Arduino and Processing Integration).
